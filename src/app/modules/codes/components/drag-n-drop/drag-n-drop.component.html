Drag And Drop

Directives to transform any tag in a draggable and/or droppable elements. Working with data model and every "JSON parsable element (i.e, Cat and Dog models). This implementation provides validation capability to check if dragged object is compliant with what the droppable element is expecting to receive. Users can also provide a callback to transform dragged object before receiving the DropEvent, with the object inside.

Draggable Directive

To make an object draggable, it has to implement DraggableType<T>.
