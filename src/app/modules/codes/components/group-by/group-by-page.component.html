Generic Array Grouping

Pipe to group any array of objects, either by a given property's value or by providing a callback able to categorize elements of the provided list.

Usage

You have 2 options to group items of your list:

  • Provide a property name, groups will be built by string representation of values.
  • Provide a callback able to receive one element of your list and match it to a list of labels.

PS: to group items inside an Object, Angular provide orderBy pipe.
